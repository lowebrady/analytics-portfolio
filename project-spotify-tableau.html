09 — Interactive Visualization / Tableau

Billboard & Spotify
Music Analysis

Built interactive Tableau visualizations to explore genre trends, track popularity, and audio feature relationships across Billboard and Spotify datasets — using dynamic parameters to let users compare any combination of musical attributes and surface what actually drives chart performance.

Role Data Visualization Analyst
Tools Tableau, Parameters, Calculated Fields
Datasets Billboard Hot 100, Spotify Audio Features API
Context Global Career Accelerator — MSU
01 — The Analytical Question

What separates a chart-topper from everything else?

Spotify's audio features API provides a unique window into the measurable properties of music: danceability, energy, valence (positivity), tempo, acousticness, speechiness, and more. Billboard's Hot 100 tells us which songs actually broke through to mainstream success. Combining the two creates an unusually rich analytical dataset — one that lets you ask whether the acoustic properties of a song predict its commercial performance.

The analytical challenge with this kind of dataset isn't the math — it's the exploration. There are dozens of possible attribute combinations to examine, multiple genre segments to compare, and trend lines that shift year to year. A static report can only answer the questions you thought to ask when you wrote it. The goal was to build a tool that made exploration feel effortless — so any question could be answered in seconds rather than requiring a new analysis.

"Spotify's audio features turn something as subjective as music into a structured dataset. The interesting question isn't whether the numbers correlate — it's whether the correlations tell you anything a music executive could actually act on."

02 — Visualization Design

Building for exploration, not just presentation

  • 01
    Genre Proportion View Designed a dynamic stacked proportion chart showing how the genre composition of the Billboard Hot 100 has shifted over time. A year-range slider lets users isolate any time window and watch genre shares expand or contract. This makes trend shifts — the rise of hip-hop, the decline of rock — immediately visible without requiring the user to compare separate snapshots.
  • 02
    Audio Feature Comparison Tool Built a parameter-driven scatter plot where both the X and Y axes are selectable from a dropdown of Spotify audio features. Users can instantly switch between "danceability vs. energy," "valence vs. tempo," or any other pairing — turning a fixed chart into a flexible exploration tool. Color encodes genre, making cluster patterns visible across comparisons.
  • 03
    Popularity vs. Audio Feature Regressions Built individual views plotting each audio feature against Billboard chart position, with a linear trend line to surface the direction and strength of each relationship. This answered the core question directly: which audio features, if any, are positively associated with chart success — and which show no meaningful relationship.
  • 04
    Genre Deep-Dives Created a genre filter that segments the full dataset so users can apply the audio feature comparison tool within a single genre — answering a more nuanced question: within hip-hop, what separates charting tracks from non-charting ones? This kind of within-genre analysis often surfaces patterns that get washed out in the full-dataset view.
  • 05
    Visual Design & Consistency Applied a consistent color palette across all views, with genre colors staying fixed regardless of which chart or filter is active. This allows users to build visual intuition about genre clusters as they explore — they don't have to re-learn the color coding on every sheet.
03 — Key Findings

What the music data actually showed

↑ Dance

Danceability Positively Correlated

Higher danceability scores showed a consistent positive association with chart performance across most genres — one of the stronger audio feature predictors in the dataset.

Genre

Genre Clusters Clearly Defined

The audio feature scatter plots revealed tight, distinct clusters by genre — hip-hop, country, and pop each occupy measurably different regions of the feature space, confirming that Spotify's attributes capture real musical differences.

Shift

Genre Share Shifts Over Time

The genre proportion view made visible a decade-long shift in Hot 100 composition — hip-hop's share grew substantially while rock declined, with pop remaining relatively stable as the consistent baseline genre.

Weak

Acousticness Not Predictive

Acousticness showed minimal correlation with chart performance — high-acoustic and low-acoustic tracks appear at all chart positions, suggesting this attribute doesn't drive commercial success on its own.

The most important methodological finding: full-dataset correlations often masked genre-specific patterns. An audio feature that appeared weakly correlated in the aggregate became a strong predictor within a single genre. This underscored the importance of segmented analysis — and validated the design decision to build genre filtering into the tool from the start.

04 — What This Project Demonstrates

Analytical and design skills applied

  • ✓
    Parameter-driven interactivity The axis-selection parameters on the scatter plot are the core design decision of this project. Instead of showing 28 separate charts (one per feature pair), a single dynamic view covers all combinations — making the tool dramatically more useful without adding visual clutter.
  • ✓
    Choosing the right visual for the question Proportion charts for composition over time. Scatter plots for relationship exploration. Regression lines for trend direction. Each view was chosen because it was the clearest way to answer the specific question that view was designed for.
  • ✓
    Thinking about the end user The design prioritized a music industry professional as the primary user — someone who wants to understand what drives chart performance, not someone who wants to explore data for its own sake. Every design decision was made with that use case in mind.
